# Obsy Agency — Awarded-site Clone
A pixel‑perfect front-end clone of the Obys Agency landing experience.
This project demonstrates advanced scroll-driven interactions, animated hero sections, multimedia presentation and a responsive project gallery.

## Highlights
- Smooth parallax scrolling (Locomotive Scroll)
- Robust animations (GSAP + ScrollTrigger)
- Interactive elements (custom loader, video preview, cursor effects)
- 3D-ready (Three.js included) and UI utilities (Shery.js)
- Lightweight, static HTML/CSS/JS — easy to deploy

## Tech
HTML • CSS • JavaScript • GSAP • ScrollTrigger • Locomotive Scroll • Three.js • Shery.js • Remixicon

## Local development
1. Clone the repo:
   git clone git@github.com:mayank-rawat-17.github.io/Awarded-site-clone/
2. Open the project folder in VS Code.
3. Serve files (option A — Live Server extension) or using Python (PowerShell):
   python -m http.server 3000
4. Open http://localhost:3000 in your browser.
